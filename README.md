# Reading Summaries Repository

This repository contains structured summaries of course readings related to
**Knowledge Graphs** and **Natural Language Processing**.

Each summary is stored as an individual Markdown file (`part1.md` through
`part5.md`). Every file includes:

- A concise summary of the assigned readings
- A references section

## Repository Structure

```text
.
├── README.md
└── summaries
    ├── part1.md
    ├── part2.md
    ├── part3.md
    ├── part4.md
    └── part5.md
```

## File Overview

- `summaries/part1.md` — Knowledge Graphs (Week 1)
- `summaries/part2.md` — Knowledge Graphs (Week 2)
- `summaries/part3.md` — Knowledge Graphs (Week 3)
- `summaries/part4.md` — Natural Language Processing (Week 1)
- `summaries/part5.md` — Natural Language Processing (Week 2)
